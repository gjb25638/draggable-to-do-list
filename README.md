# draggable-to-do-list

可拖拉雙層to-do-list

# 環境需求
- Node 18.12.1
- Git
- pnpm
- vscode
- Volar

# 安裝 pnpm
```PowerShell
npm i -g pnpm
```

# 安裝 node 依賴模組
```PowerShell
pnpm i
```

# 開發環境
```PowerShell
git checkout master
pnpm serve
```

# 輸出生產
```PowerShell
pnpm build
```

# docker 環境
```PowerShell
git checkout add_docker
make up
```
